# AI-Generated Images

This directory contains AI-generated images for the Lawyer Free application.

## Generating Images

Run the following command from the project root:

```bash
OPENAI_API_KEY=sk-your-key-here node scripts/generate-images.js
```

## Available Images

After generation, the following images will be available:
