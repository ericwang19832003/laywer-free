## Task 1: Add DRAFT watermark to generated documents

Downloaded documents should include a visible DRAFT watermark header to signal they haven't been reviewed.

**Files:**
- Modify: `apps/web/src/app/api/document-generation/route.ts`

The generated document text is returned as `data.document`. Prepend a header before returning it.

**Step 1: Add watermark to generated document text**

Find the successful return (around line 200–214):
```typescript
return NextResponse.json({
  success: true,
  document,
  ...
})
```

Before the return, add:
```typescript
const watermark = [
  '═══════════════════════════════════════════════════════════════',
  'DRAFT — NOT REVIEWED BY A LICENSED ATTORNEY',
  'This document was generated by AI and has not been reviewed,',
  'approved, or filed. Do not use without independent legal review.',
  '═══════════════════════════════════════════════════════════════',
  '',
  '',
].join('\n')

const documentWithWatermark = watermark + document
```

Change the return to use `documentWithWatermark`:
```typescript
return NextResponse.json({
  success: true,
  document: documentWithWatermark,
  ...
})
```
